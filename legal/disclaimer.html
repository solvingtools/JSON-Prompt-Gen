1. AI Generation Limitations

AI Video Generation is experimental and unpredictable.

The structured JSON prompts provided by this tool are optimized based on current model documentation and community best practices. However:

  • Results Vary: The exact same JSON input can produce vastly different videos each time you run it.
  • Model Hallucinations: AI models may sometimes ignore specific camera instructions (like "dolly zoom") or lighting parameters.
  • Output Quality: We cannot guarantee that the resulting video will be high-definition, glitch-free, or physically accurate.

2. Platform Specifics

Features in this tool are tailored to specific platforms, but these platforms change rapidly:

  • Google Veo: Optimization is based on the Veo API and Gemini video capabilities.
  • OpenAI Sora: Prompts are structured for the Sora research preview/API.

If a platform updates its API or capabilities, parts of our prompt generator may become temporarily outdated before we can release a fix.
